# GitHub 发布简介 / GitHub Release Intro

## 中文

**服装管理后台系统** 是一个面向服装拍摄现场、样衣间和小型门店的轻量级 Web 后台。它提供商品资料管理、Look（造型/套装）管理、图片上传与缩略图、SQLite 本地存储、Excel 导入导出、扫码友好的搜索体验，以及前端中英文切换。

技术栈：Node.js + Express + SQLite + Vue 3 CDN + 原生 CSS。无需前端构建流程，适合单机或单服务器快速部署。

快速启动：

```bash
npm install
npm start
```

默认访问：`http://localhost:8080`

## English

**Clothing Management System** is a lightweight web admin for clothing shoots, sample rooms, and small retail workflows. It includes product records, Look/outfit management, image uploads with thumbnails, local SQLite storage, Excel import/export, barcode-friendly search, and a Chinese/English UI switcher.

Stack: Node.js + Express + SQLite + Vue 3 via CDN + plain CSS. No frontend build step is required, making it easy to deploy on a single machine or server.

Quick start:

```bash
npm install
npm start
```

Default URL: `http://localhost:8080`
